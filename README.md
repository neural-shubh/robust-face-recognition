# 🥷 The Disguise-Resistant Face & Person Identifier

An AI-powered Computer Vision system capable of identifying people even when facial appearance changes due to disguises such as masks, sunglasses, caps, beards, scarves, or partial occlusions.

This project combines deep learning-based face representation with robust feature extraction techniques to improve person recognition under challenging real-world conditions.

---

## 🚀 Features

- Face Detection
- Robust Face Embedding Generation
- Recognition under Partial Occlusion
- Disguise-Resistant Identification
- Image Similarity Matching
- Confidence Score Prediction
- Works on unseen images
- Easy to extend for real-time webcam applications

---

## 📌 Problem Statement

Traditional face recognition systems fail when people intentionally or unintentionally alter their appearance using:

- 😷 Face masks
- 🧢 Caps
- 🕶 Sunglasses
- 🧔 Beard
- 🧣 Scarves
- Partial Occlusion
- Lighting Variations

This project aims to build a recognition pipeline that remains reliable under such challenging conditions.
